# Scuderia Frontend

## Install

- Required engines:

  ```json
  {
    "node": "18",
    "pnpm": "8.6.10"
  }
  ```

- (OPTIONAL) On MacOS, install [volta-cli](https://volta.sh/ "volta")
- Run

  ```bash
  pnpm install
  pnpm dev
  ```
